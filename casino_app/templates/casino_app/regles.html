Règles du jeu – Roulette

La roulette est un jeu de hasard consistant à miser sur le numéro ou la couleur sur lesquels la bille s’arrêtera après le lancement de la roue. Cette version de la roulette comporte un seul zéro (0).

La roue

  • La roulette est composée de 37 numéros : de 0 à 36.
  • Le 0 est de couleur verte.
  • Les autres numéros sont répartis entre le rouge et le noir.
  • La bille tourne dans le sens inverse de la roue.

Les mises autorisées

Les joueurs peuvent placer leurs jetons uniquement sur les cases. Les mises entre deux cases ou sur les lignes ne sont pas autorisées.

  • Mise sur un numéro plein (0 à 36)
  • Mise sur la couleur : rouge ou noir
  • Mise sur pair ou impair
  • Mise sur manque (1–18) ou passe (19–36)

Valeur des jetons

Les jetons disponibles ont les valeurs suivantes :

  • Jeton de 1
  • Jeton de 5
  • Jeton de 10
  • Jeton de 25

Déroulement d’un tour

  1. Le joueur place ses jetons sur les cases de son choix.
  2. Une fois les mises terminées, la roulette est lancée.
  3. La bille s’arrête sur un numéro gagnant.
  4. Les gains sont calculés selon les mises placées.

Paiements

  • Numéro plein : gain de 35 fois la mise
  • Couleur / Pair–Impair / Manque–Passe : gain de 1 fois la mise

Important

Si la bille tombe sur le 0, toutes les mises sur les couleurs, pair/impair et manque/passe sont perdues.
